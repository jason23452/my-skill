# Typeset

`typeset` 用於改善 typography hierarchy、font pairing、line length、wrapping 與內容可讀性。

## 流程

1. 檢查現有 font stack 與載入方式。
2. 定義 display、body、utility、code 的字體角色。
3. 建立 type scale、line-height、letter-spacing。
4. 修正 headings、body copy、labels、buttons、captions。
5. 測試 mobile 與長文字。

## 規則

- 內文行長 65-75ch。
- 不配兩個很像的 sans-serif。
- 大標題不要大到 overflow。
- h1-h3 可用 `text-wrap: balance`。
- 長文可用 `text-wrap: pretty`。

## 完成標準

文字層級清楚、閱讀舒適、品牌個性更明確，且不同 viewport 不爆版。
