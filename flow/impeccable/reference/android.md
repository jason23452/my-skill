# Android 參考

原生 Android UI 應遵守 Material Design 3 的結構、導覽、主題化與互動慣例。

## 原則

- 使用 Material color roles、type scale、shape、motion。
- 遵守 window size classes 與 adaptive layout。
- 支援 dark theme，不要只反相。
- 保持 back behavior、system bars、edge-to-edge 與 accessibility。

## 版面

- Phone：bottom navigation、navigation bar、single pane。
- Tablet / foldable：navigation rail、two pane、list-detail。
- Large screen：善用寬度，不要只是放大的 phone UI。

## 完成標準

介面符合 Android 使用者預期，並在不同 window size class 下自然適配。
