# Brand Register

當 design 本身就是產品時使用：brand sites、landing pages、marketing pages、campaign pages、portfolio、long-form content、about pages。

## 原則

- 訪客的第一印象就是交付物。
- 品牌頁面可以更 committed、full palette 或 drenched，不必永遠 restrained。
- 色彩、字體、layout、imagery 必須來自題材世界。
- 不要用通用 SaaS hero、三張 feature cards、假 metrics 當預設。

## 版面

- Hero 必須提出視覺論點。
- Section rhythm 要變化，不要每段同樣 spacing。
- 如果 brief 暗示 imagery，就應該有 imagery；不要用抽象 CSS panel 假裝。
- Cards 只有在真的支持內容時才用。

## 完成標準

頁面具有明確品牌觀點，訪客能記住它，而不是只覺得「乾淨現代」。
