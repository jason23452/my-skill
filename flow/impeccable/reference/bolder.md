# Bolder

`bolder` 用於讓安全、平淡或模板感太重的設計更有力。Bold 不是更多漸層、更多 neon、更多 glassmorphism，而是更明確的取捨。

## 流程

1. 先檢查父 skill 的 AI slop 禁止清單。
2. 找出目前設計太弱的原因：低對比、缺少主張、所有元素同權重、內容節奏單一。
3. 在既有 design system 內放大視覺語言。
4. 強化 typography、composition、color commitment、surface contrast 或 motion moment。
5. 保持 coherence，不要加入不屬於產品世界的效果。

## 完成標準

畫面更有記憶點與主張，但仍清楚、可用、符合品牌。
