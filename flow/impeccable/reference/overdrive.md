# Overdrive

`overdrive` 用於把視覺或互動推到更大膽、更技術性的方向。只在使用者明確要高衝擊或實驗性時使用。

## 流程

1. 確認使用者真的要高強度設計，不是一般 polish。
2. 選擇一個主要技術或視覺概念：WebGL、canvas、mask、clip-path、advanced motion、immersive layout。
3. 控制範圍，避免整頁每個元素都在競爭。
4. 實作 fallback、reduced motion 與 performance guard。
5. 驗證核心內容仍可讀、可操作。

## 完成標準

效果令人印象深刻，但不犧牲可用性與效能。
